# Диагностика проблем

## Сервер не поднимается

Проверьте:

1. Команду запуска `poetry run python run_server.py`.
2. Наличие и корректность `configs/server.toml`.
3. Не занят ли HTTP-порт другим процессом.

## Клиент запущен, но цель не держится

Проверьте:

1. Правильный ли выбран пресет.
2. Не слишком ли высокий `contrast_threshold` для слабой цели.
3. Не слишком ли узкая зона поиска (`min_gate`, `max_gate`).

## Цель быстро уходит в LOST

Проверьте:

1. `max_lost_frames` в пресете.
2. Параметры `target_recovery`.
3. Качество исходного видео (резкость, шум, пересвет).
