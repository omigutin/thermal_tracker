# Быстрый старт

Этот документ для первого запуска без глубокого погружения в код.

## Что делает система

1. Получает кадры (из видео, потока или shared memory).
2. Улучшает кадр для анализа.
3. Ищет и сопровождает цель.
4. Показывает результат в интерфейсе.

## Минимальный порядок запуска

### 1) Запустить сервер

```bash
poetry run python run_server.py
```

Это поднимет gateway и runtime worker в режиме `all`.

### 2) Запустить desktop-клиент

```bash
poetry run python run_desktop_client.py
```

Desktop-клиент нужен для визуальной проверки и ручного выбора цели.

### 3) (Опционально) Открыть web-клиент

```bash
poetry run python run_web_client.py
```

## Что делать после запуска

1. Выбрать пресет.
2. Начать подачу кадров.
3. Кликнуть по цели в кадре.
4. Проверить, удерживается ли цель при движении и кратковременной потере.

## Если что-то не работает

Перейдите в [Диагностика проблем](troubleshooting.md).
