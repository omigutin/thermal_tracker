# Стадии обработки

Ниже описан путь одного кадра простыми словами.

## 1. Подготовка кадра (preprocessing)

Кадр очищается от шума и приводится к удобному виду для алгоритмов.

## 2. Оценка движения камеры (global motion)

Система оценивает, насколько сместился кадр из-за движения камеры.
Это нужно, чтобы не путать движение камеры с движением цели.

## 3. Сопровождение цели (target tracking)

Если цель уже выбрана кликом, трекер обновляет её положение на новом кадре.

## 4. Восстановление цели (target recovery)

Если цель временно пропала, система пытается найти её рядом с последней позицией.

## 5. Переходы состояния

Основные состояния:

- `IDLE` — цель ещё не выбрана;
- `TRACKING` — цель уверенно сопровождается;
- `SEARCHING` — цель временно потеряна, идёт попытка вернуть;
- `LOST` — цель не удалось вернуть, нужен новый клик.
