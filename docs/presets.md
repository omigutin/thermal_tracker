# Пресеты

Пресет — это готовый профиль настроек для конкретной ситуации.

## Что внутри пресета

Обычно есть секции:

- `preprocessing` — подготовка кадра;
- `global_motion` — секция настроек для стадии `frame_stabilization` (оценка сдвига камеры);
- `target_recovery` — восстановление после потери;
- `click_selection` — стартовая инициализация по клику;
- `opencv_tracking` или `irst_tracking` — параметры трекера.

## Новый пресет для маленькой цели

`presets/irst_small_target.toml` предназначен для маленьких тепловых объектов,
где обычный шаблонный подход теряет устойчивость.

Ключевая идея: контрастный поиск цели + фильтр Калмана.

Подробно: [IRST-трекинг](irst_tracking.md).

## Практический выбор

- Если цель маленькая и фон сложный — начните с `irst_small_target`.
- Если есть стабильный контрастный патч без сильного blur — можно пробовать классический OpenCV-путь.
