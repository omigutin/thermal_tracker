# IRST-трекинг

IRST (Infrared Search and Track) — это подход для сопровождения тепловой цели,
когда объект маленький и обычный шаблонный трекинг нестабилен.

## Простое объяснение

Система ищет локально контрастный "тепловой сгусток" рядом с ожидаемым местом цели,
а затем сглаживает движение прогнозом (фильтр Калмана).

## Когда полезно

- цель очень маленькая;
- изображение периодически размывается;
- фон текстурный, и шаблонный трекер "плавает".

## Ключевые параметры

В `irst_tracking` обычно настраивают:

- `contrast_threshold` — насколько цель должна отличаться от фона;
- `min_blob_area` / `max_blob_area` — размер допустимого объекта;
- `min_gate` / `gate_growth` / `max_gate` — радиус зоны поиска;
- `max_lost_frames` — сколько кадров ждать до признания потери.

## Связь с recovery

Для согласованной работы рекомендуется `target_recovery.method = "irst_contrast"`.
